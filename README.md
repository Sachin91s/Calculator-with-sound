# ✨ Features

- ➕ **Basic Arithmetic Operations** (+, −, ×, ÷)
- 🧮 **Parentheses Support** for complex calculations
- ❌ **Clear (C)** and **Delete (del)** functions
- 🎵 **Button Click Sound Effect**
- 🎨 **Clean & Modern UI**
- 📱 **Fully Responsive Design**
